rpg stuff

RPG Stuff is my own personal project that is in development. The website contains resources to be used with tabletop roleplaying games, a personal hobby of mine. Games aside, it is a good representation of various ways that I can include interactivity into a website. This website is still under development, but currently contains a random story generator using icons from game-icons.net.




the armory

The Armory is an employee portal that I built for my current day job. It is used by 80+ employees of ARM Group Inc. and offers an announcement feed, a calendar for work events, profiles for each individual employee, a phone list, and more. I do not have a live demo of the site available, as it is only accessible by employees, but most of the code is availble on GitHub.
